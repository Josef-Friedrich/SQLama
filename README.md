# SQLama

## SQL in the browser

* https://github.com/sqlpad/sqlpad
* https://sqlime.org
* https://github.com/zzzprojects/sqlfiddle2

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
